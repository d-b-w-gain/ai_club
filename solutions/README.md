# Solutions

This folder is for outcomes from club challenges.

Use GitHub issues for project ideas, discussion, and the backlog. Use this folder when someone has something worth keeping after a meeting:

- a prompt that worked well
- a demo or screenshot
- a short write-up of an approach
- notes on what failed
- links to useful examples
- code, files, or other artefacts the club wants to keep

Each solution can link back to the issue that started it.
