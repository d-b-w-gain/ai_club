# Open-Source CAD Collaboration Project

## Idea

Present or trial an open-source collaborative CAD project as a practical example of AI-adjacent technical work.

The aim is not to make the club about CAD. It is a real project that can help start a broader conversation about how AI and other tools fit into actual creative and technical workflows.

## Questions

- What is the simplest way for members to access it?
- Can multiple people view or edit the same project?
- Where could AI help this kind of workflow?
- Where would AI be unhelpful, distracting, or risky?
- What makes a tool genuinely useful to people working together?

## Next Step

Prepare a short presentation, demo, or shared trial that the group can use as a discussion point.
